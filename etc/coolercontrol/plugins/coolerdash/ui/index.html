Authentication — CC4 (Bearer Token)

CoolerControl 4.0 requires API authentication.
  1. Open the CoolerControl UI
  2. Bottom-left corner → 🔒 Access Protection → Access Tokens
  3. Generate a new Access Token
  4. Copy your token key (cc_…)
  5. Paste the token in the Access Token field below
Enter a CC4 Bearer token (cc_…). After saving, the token is stored securely in coolercontrol.json (chmod 600) — this field will appear empty on the next load.
